# План повторений по кривой Эббингауза для изучения иностранных слов

Этот план помогает закреплять новые слова в долговременной памяти, повторяя их по оптимальным интервалам.

| День | Что делать | Объяснение |
|------|-----------|------------|
| День 1 | Изучаешь 10 новых слов (Список №1) | Первичное знакомство с материалом |
| Через 30 минут | Повторяешь Список №1 | Быстрое закрепление, пока память свежая |
| День 2 | Изучаешь Список №2 (новые 10 слов) + повторяешь Список №1 | Первое повторение, чтобы предотвратить забывание |
| День 3 | Список №3 (новые 10 слов) + повтор Списков №1 и №2 | Повторение через 2 дня после изучения первых слов |
| День 4 | Список №4 (новые 10 слов) + повтор Списков №1, №2, №3 | Закрепление ранних слов, добавление новых |
| День 8 | Повторяешь Списки №1–4 | Через неделю слова переходят в долговременную память |
| День 15 | Повторяешь всё | Финальное закрепление, проверка усвоения |

## Примечания
- Каждый “Список” — это 10 новых слов.
- После двух недель слова, которые хорошо запомнились, можно считать освоенными.
